# TODO APP
The todo app is a fullstack app that keeps track of the users planned tasks. This app is used to give the user record of the task which has been inputed.
## Features
The todo app features key fullstack application functionalities. This functionality keeps the app secure and exclusive to different users.
The application contains features such as:
1. Authentication: This separates every users data from each other making each user have exclusive access to their data.
2. Cloud storage: This application features cloud storage making the users data to be persistent on the cloud and preventing data loss.
3. CRUD: The app allows the user to creat tasks, modify the stipulated task and delete this task as the user wishes.
4. Confirmation: This application allows the user to verify the actions which they want to execute. The user will get a confirmation if they want to delete and modify their necessary data.
